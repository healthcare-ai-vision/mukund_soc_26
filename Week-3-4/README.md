# Week 3-4: Object Detection using YOLO

## Dataset
- **Dataset:** Chess Pieces Detection
- **Source:** https://universe.roboflow.com/joseph-nelson/chess-pieces-new
- **Classes:** 12 chess piece classes (King, Queen, Rook, Bishop, Knight, Pawn for both black and white)

## Model
- **Framework:** YOLO11
- **Model:** `yolo11n.pt`

## Training Details
- Epochs: 10
- Image Size: 640 × 640
- Batch Size: 16
- Platform: Google Colab (GPU)

## Tasks Performed
- Downloaded dataset from Roboflow Universe
- Trained YOLO object detection model
- Validated model performance
- Performed predictions on test images
