# 🏥 AI Healthcare Assistant

An AI-powered web application for **skin disease detection** using **Deep Learning**, **Explainable AI**, and **Large Language Models (LLMs)**.

This project was developed as part of the **Summer of Code (SOC) 2025, IIT Bombay** under the **AI for Healthcare** domain.

---

## 📌 Features

- 🔍 Skin disease classification from dermoscopic images
- 🧠 EfficientNet-B0 deep learning model
- 📊 Top-3 disease predictions with confidence scores
- 🔥 Grad-CAM visualization for model explainability
- 🤖 AI-generated medical explanation using OpenRouter
- 🌐 Interactive web interface built with Streamlit

---

## 📂 Dataset

- **Dataset:** ISIC 2019 Skin Lesion Dataset
- **Total Images:** 25,331
- **Disease Classes:** 8

Classes:
- Actinic Keratosis (AK)
- Basal Cell Carcinoma (BCC)
- Benign Keratosis (BKL)
- Dermatofibroma (DF)
- Melanoma (MEL)
- Melanocytic Nevus (NV)
- Squamous Cell Carcinoma (SCC)
- Vascular Lesion (VASC)
